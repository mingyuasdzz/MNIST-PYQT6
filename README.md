# 手写数字识别

基于PyTorch和PyQt6(ai实现)的手写数字识别应用，使用卷积神经网络(CNN)实现MNIST数字识别。

## 功能特点

- 手写数字输入界面
- 实时数字识别
- 显示识别置信度
- Top-3预测结果展示
- 支持GPU加速(CUDA)

## 项目结构

```
shouxieshuzishibie/
├── main.py          # 主程序，包含GUI界面和识别逻辑
├── cnn_bn.py        # CNN模型定义和训练代码
├── model.pth        # 训练好的模型权重
└── requirements.txt # 项目依赖
```

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 运行识别程序

```bash
python main.py
```

### 训练模型(可选)

如果需要重新训练模型：

```bash
python cnn_bn.py
```

训练完成后，模型会自动保存为`model.pth`。

## 技术细节

### 模型架构

```
输入层 (1x28x28)
    ↓
卷积层1 (Conv2d: 1→32, kernel=3x3, padding=1)
    ↓
批归一化1 (BatchNorm2d: 32)
    ↓
ReLU激活
    ↓
最大池化1 (MaxPool2d: 2x2) → (32x14x14)
    ↓
卷积层2 (Conv2d: 32→64, kernel=3x3, padding=1)
    ↓
批归一化2 (BatchNorm2d: 64)
    ↓
ReLU激活
    ↓
最大池化2 (MaxPool2d: 2x2) → (64x7x7)
    ↓
展平 (Flatten) → (64x7x7 = 3136)
    ↓
全连接层1 (Linear: 3136→128)
    ↓
批归一化3 (BatchNorm1d: 128)
    ↓
ReLU激活
    ↓
全连接层2 (Linear: 128→10)
    ↓
输出层 (10个类别)
```

### 模型组件说明

1. **卷积层 (Conv2d)**
   - 第一层：32个3x3卷积核，提取局部特征
   - 第二层：64个3x3卷积核，提取更复杂的特征

2. **批归一化 (Batch Normalization)**
   - 加速训练收敛
   - 提高模型稳定性
   - 防止梯度消失/爆炸

3. **池化层 (Max Pooling)**
   - 降低特征图尺寸
   - 减少计算量
   - 增强特征不变性

4. **全连接层**
   - 第一层：3136→128，特征整合
   - 第二层：128→10，分类输出



### 训练过程

1. **数据准备**
   - 下载MNIST数据集
   - 归一化到[0,1]范围
   - 批次大小：32

2. **前向传播**
   - 输入图像通过各层网络
   - 计算预测输出

3. **损失计算**
   - 使用交叉熵损失函数
   - 计算预测与真实标签的差异

4. **反向传播**
   - 计算梯度
   - 使用Adam优化器更新参数
   - 学习率：0.001

5. **模型评估**
   - 在测试集上评估准确率
   - 保存最佳模型

### 图像处理

1. 自动检测数字边界框
2. 添加12%的padding
3. 将数字居中到280x280画布
4. 缩放到28x28
5. 反色处理(白底黑字 -> 黑底白字)

### 性能

- 在MNIST测试集上准确率约99%
- 支持实时识别
- GPU加速可显著提升推理速度

## 注意事项

- 确保手写数字清晰，笔画连贯
- 在280x280的画布中央书写
- 使用黑色笔画在白色背景上书写
